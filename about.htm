PlotSQLite - Midvatten plot creator

An application for creating custom time series plots from data stored in a sqlite database.

Version: 0.1
Author: Josef Källgården (groundwatergis@gmail.com)
More info: http://sourceforge.net/projects/plotsqlite/

There is also a closely related project: http://sites.google.com/site/midvattenpluginforqgis/
